### 何时部署爬虫

当你有自己的 Windows 云服务器后，可以部署爬虫实现：
- 自动抓取论坛比赛结果
- 定时同步到简道云
- 减少人工录入工作量

### 部署步骤

1. 将 `fm26-league-crawler` 上传到服务器
2. 安装 Python 和依赖
3. 配置论坛账号和简道云 API
4. 设置定时任务（每天 22:00）

详见：`爬虫扩展指南.md`

## 🏗️ 技术架构

### 数据流

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│  网页录入   │     │  爬虫抓取    │     │  人工录入   │
│  (手动)     │     │  (自动)      │     │  (备用)     │
└──────┬──────┘     └──────┬───────┘     └──────┬──────┘
       │                   │                    │
       └───────────────────┼────────────────────┘
                           ↓
                 ┌─────────────────┐
                 │   简道云 API    │
                 │  (统一存储)     │
                 └────────┬────────┘
                          ↓
                 ┌─────────────────┐
                 │   网页展示      │
                 │  (实时读取)     │
                 └─────────────────┘
```

### 核心设计原则

| 原则 | 说明 |
|------|------|
| **数据统一** | 网页和爬虫都写入简道云同一张表 |
| **来源区分** | 通过「数据来源」字段区分手动/爬虫 |
| **状态管理** | 爬虫数据「待审核」，手动录入「已确认」 |
| **无缝扩展** | 增加爬虫无需修改网页代码 |

## 🆘 快速故障排查

### 网页打不开

- 检查是否用浏览器打开（不是记事本）
- 尝试 Chrome/Edge/Firefox

### 配置保存失败

- 检查浏览器是否禁用 localStorage
- 清除浏览器缓存后重试

### 数据提交失败

1. 检查 API 配置是否正确
2. 检查表 ID 是否对应
3. 查看浏览器控制台（F12）错误信息

### 爬虫运行失败

1. 检查 Python 是否安装
2. 检查依赖是否安装成功
3. 查看 `crawler.log` 日志
