# 更新日志

## v1.1 - 2026-04-08

### 🎉 新增功能

#### 1. 🏆 杯赛积分榜
- 支持杯赛小组赛和淘汰赛
- 可按组别筛选（A 组/B 组/C 组/D 组/淘汰赛）
- 自动计算积分、排名、净胜球
- 独立于联赛积分榜

#### 2. 🏆 杯赛赛程
- 支持杯赛所有阶段（小组赛/16 强/8 强/4 强/决赛）
- 可按阶段筛选查看
- 显示组别信息
- 状态标签（已确认/待审核）

#### 3. ⚽ 比赛录入增强
- 新增比赛类型选择（联赛/杯赛）
- 选择杯赛时自动显示阶段和组别字段
- 选择联赛时显示轮次字段
- 智能切换表单字段

#### 4. ⚙️ 配置增强
- 新增杯赛记录表 ID 配置
- 支持联赛和杯赛分离存储

### 📋 简道云配置更新

#### 需要新增：杯赛记录表

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 阶段 | 单选 | 小组赛/16 强/8 强/4 强/决赛 |
| 组别 | 单选 | A 组/B 组/C 组/D 组（淘汰赛可留空） |
| 主队 | 单行文本 | 主队名称 |
| 客队 | 单行文本 | 客队名称 |
| 主队得分 | 数字 | 主队进球 |
| 客队得分 | 数字 | 客队进球 |
| 比赛日期 | 日期 | 比赛时间 |
| 状态 | 单选 | 待审核/已确认 |
| 数据来源 | 单选 | 人工录入/论坛爬虫 |
| 录入人 | 单行文本 | 录入者 |
| 备注 | 多行文本 | 额外信息 |

#### 配置步骤

1. 在简道云创建新表：`杯赛记录表`
2. 按上方结构添加字段
3. 复制表 ID（类似 `TBxxxxxxxx`）
4. 在网页「设置」页面填入「杯赛记录表 ID」
5. 点击「保存配置」→「测试连接」

### 📖 使用指南

#### 录入联赛比赛

1. 点击「比赛录入」
2. 比赛类型选择「联赛」
3. 填写轮次、球队、比分
4. 提交

#### 录入杯赛比赛

1. 点击「比赛录入」
2. 比赛类型选择「杯赛」
3. 选择阶段（小组赛/16 强/8 强/4 强/决赛）
4. 选择组别（小组赛必填，淘汰赛可选）
5. 填写球队、比分
6. 提交

#### 查看杯赛积分榜

1. 点击「🏆 杯赛」
2. 使用组别筛选查看不同组
3. 选择「淘汰赛」查看淘汰赛阶段

#### 查看杯赛赛程

1. 点击「🏆 杯赛赛程」
2. 使用阶段筛选查看不同阶段
3. 可查看已结束比赛的比分

### ⚠️ 注意事项

1. **杯赛表是独立的**
   - 联赛数据在「比赛记录表」
   - 杯赛数据在「杯赛记录表」
   - 两者互不影响

2. **组别字段说明**
   - 小组赛：必须选择组别（A/B/C/D 组）
   - 淘汰赛：组别可留空或选择「淘汰赛」

3. **积分计算**
   - 杯赛和联赛积分分开计算
   - 杯赛积分榜只显示杯赛数据
   - 联赛积分榜只显示联赛数据
