# FM26 联赛管理系统 - 使用说明

**深色模式 · 免登录 · 双击即用**

---

## 🚀 快速开始（3 分钟）

### 第 1 步：打开网页

双击 `index.html` 文件，浏览器会自动打开。

> 💡 推荐使用 Chrome、Edge 或 Firefox

---

### 第 2 步：配置简道云 API

1. 点击导航栏的 **「⚙️ 设置」**
2. 填写以下信息（从简道云获取）：

| 配置项 | 获取方式 |
|--------|----------|
| 简道云 App ID | 简道云应用设置 → API 集成 |
| 简道云 API Key | 简道云应用设置 → API 集成 |
| 比赛记录表 ID | 比赛记录表 → 表设置 |
| 球队信息表 ID | 球队信息表 → 表设置 |
| 交易记录表 ID | 交易记录表 → 表设置 |
| 公告表 ID | 公告表 → 表设置 |

3. 点击 **「保存配置」**
4. 点击 **「测试连接」** 确认配置正确

---

### 第 3 步：开始使用

| 功能 | 操作 |
|------|------|
| 📊 查看积分榜 | 点击「积分榜」标签 |
| 🏆 杯赛积分榜 | 点击「杯赛」→ 可按组别筛选 |
| ⚽ 录入比赛 | 点击「比赛录入」→ 选择联赛/杯赛 → 提交 |
| 📅 联赛赛程 | 点击「联赛赛程」→ 可按轮次筛选 |
| 🏆 杯赛赛程 | 点击「杯赛赛程」→ 可按阶段筛选 |
| 💰 提交交易 | 点击「交易」→ 填写申请 → 提交 |
| 📢 查看公告 | 点击「公告」 |

## 📋 功能说明

### 1. 积分榜

- **自动计算**：胜/平/负、积分、净胜球
- **实时排名**：按积分→净胜球→进球数排序
- **数据统计**：参赛球队数、已赛场次、总进球数

### 2. 比赛录入

- **选择球队**：从下拉列表选择主/客队
- **输入比分**：填写双方进球数
- **自动同步**：提交后立即更新积分榜
- **数据来源标记**：显示"手动"或"爬虫"

### 3. 赛程管理

- **轮次筛选**：按轮次查看比赛
- **状态显示**：未开始/进行中/已确认
- **比分展示**：已结束的比赛显示比分

### 4. 交易申请

- **球员信息**：姓名、位置
- **转会信息**：转出/转入球队、类型、金额
- **审核流程**：提交后状态为"待审核"

### 5. 公告系统

- **优先级**：普通/重要/紧急
- **发布信息**：发布人、时间
- **分类显示**：按类型展示

## 🌐 后续扩展：部署爬虫

当你有自己的云服务器后，可以部署爬虫自动抓取论坛数据。

### 步骤

1. **上传爬虫代码到服务器**
   - 使用之前生成的 `fm26-league-crawler` 代码包

2. **配置服务器环境**
   ```bash
   # 安装 Python
   # 安装依赖
   pip install -r requirements.txt
   playwright install chromium
   ```

3. **配置 config.py**
   - 填入论坛账号
   - 填入简道云 API 配置（和网页用同一个）

4. **设置定时任务**
   ```bash
   # Linux crontab
   0 22 * * * cd /path/to/crawler && python crawler.py
   ```

5. **验证数据**
   - 爬虫抓取的数据会写入简道云
   - 网页会自动显示爬虫数据（标记为"爬虫"来源）

### 注意事项

| 事项 | 说明 |
|------|------|
| 数据去重 | 爬虫会检查是否已存在相同比分 |
| 状态区分 | 爬虫数据标记为"待审核"，手动录入标记为"已确认" |
| 审核流程 | 管理员可在简道云审核爬虫数据 |

## 💡 使用技巧

### 1. 多设备同步

- 配置保存在浏览器 localStorage
- 在不同电脑使用需重新配置
- 可导出配置（未来功能）

### 2. 数据备份

- 所有数据存储在简道云
- 可在简道云后台导出 Excel
- 建议定期备份

### 3. 权限管理

- 网页无登录，所有人可录入
- 如需权限控制，在简道云设置
- 或未来添加登录功能

### 4. 手机使用

- 网页是响应式设计
- 手机浏览器打开即可用
- 可添加到主屏幕

## 🆘 常见问题

### Q: 配置保存后刷新就没了？

A: 配置保存在浏览器 localStorage，清除浏览器数据会丢失。建议截图保存配置。

### Q: 积分榜不更新？

A: 检查比赛状态是否为"已确认"，只有已确认的比赛才计入积分。

### Q: 提交失败，提示 API 错误？

A: 
1. 检查 App ID、API Key、表 ID 是否正确
2. 检查简道云 API 是否启用
3. 查看浏览器控制台（F12）的错误信息

### Q: 球队列表是空的？

A: 需要先在简道云「球队信息表」中添加球队数据。

### Q: 可以在手机上用吗？

A: 可以！网页是响应式设计，手机浏览器打开即可。
